10,000 processes

This webpage shows 10,000 independent go blocks all running at the same time. This code is based off of David Nolen's post from his blog. The code is written in Clojurescript, which is variant of Clojure/Lisp. Clojurescript transpiles into Javascript.

In a nutshell, there are 10,000 go routines (using the core.async library) which are managing the 10,000 elements of the table. Once every 10s the routine will randomly change the number and color of the table element. The table is updates every 40 ms.

Although today this table would most likely be rendered in React, this is still very cool technology!
