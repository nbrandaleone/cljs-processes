10,000 processes

10,000 independent go blocks all running at the same time. This code is based off of David Nolen's post from his blog. The code is written in Clojurescript, which is variant of Clojure/Lisp. Clojurescript transpiles into Javascript. There are some code snippets in David's original blog post.

I am going to have two follow-up posts, where I replicate this code in ELM, and in Elixir Phoenix LiveView.

In a nutshell, there are 10,000 go routines (using core.async library) which are managing the 10,000 elements of the table. Once every 10s the routine will randomly change the number and color of the table element. The table is updates every 40 ms.

This is still very cool technology!
